# Contributing Guidelines

Thank you for your interest in this project.

This repository is currently maintained as a personal research and learning project.
External contributions are not actively solicited at this stage, but feedback is welcome.

## Scope of Contributions

The following are welcome:
- Bug reports
- Architectural feedback
- Design discussions
- Simulation ideas

The following are currently out of scope:
- Major feature additions
- API redesigns
- UI/UX changes
- Production hardening

## Development Setup

Please refer to the README for environment setup and architecture overview.

## Code Style

- Follow existing patterns
- Prefer clarity over optimization
- Add comments for non-obvious logic

## Discussions and Feedback

- Use GitHub Issues for discussion
- Clearly state assumptions and context
- Keep feedback constructive and specific
